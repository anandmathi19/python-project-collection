# Typing Speed Test (Python Tkinter)

A simple **Typing Speed Test application** built using **Python and Tkinter**.
This program measures the user's **typing speed (Words Per Minute)** and **accuracy** by comparing typed text with a sample paragraph.

---

## 🚀 Features

* Start typing test with a button
* Calculates **Words Per Minute (WPM)**
* Calculates **typing accuracy**
* Simple and clean GUI
* Built using **Tkinter**

---

## 🖥️ Technologies Used

* Python
* Tkinter
* Time module

## ▶️ How to Run

1. Install Python (if not installed)

2. Download or clone this project

```
git clone https://github.com/yourusername/typing-speed-test.git
```

3. Navigate to the project folder

```
cd typing-speed-test
```

4. Run the program

```
python main.py
```

---

## 📊 How It Works

1. Click **Start Test**
2. Type the given paragraph
3. Click **Check Speed**
4. The program calculates:

   * **Typing Speed (WPM)**
   * **Accuracy**

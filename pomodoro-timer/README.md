# Pomodoro Timer 🍅

A simple Pomodoro Timer built with Python and Tkinter to help improve focus and productivity.

## Features

* 25 minute work timer
* 5 minute short break
* 20 minute long break
* Automatic session switching
* Checkmarks for completed sessions

## Built With

* Python
* Tkinter

## How It Works

The Pomodoro Technique is a time management method:

1. Work for 25 minutes
2. Take a 5 minute break
3. After 4 sessions take a longer break

This application automates the entire process.

## How to Run

Clone the repository

```
git clone https://github.com/yourusername/pomodoro-timer.git
```

Go to the project folder

```
cd pomodoro-timer
```

Run the program

```
python main.py
```

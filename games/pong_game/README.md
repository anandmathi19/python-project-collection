# Pong Game 🏓

A simple Pong game built with **Python** using the **Turtle graphics library**.

## About

This project recreates the classic arcade Pong game. Two players control paddles and try to bounce the ball past the opponent to score points.

## Project Files

* `main.py` – starts the game and handles the main loop
* `paddle.py` – controls paddle movement
* `ball.py` – manages ball movement and collisions
* `scoreboard.py` – keeps track of the score

## How to Run

1. Make sure Python is installed.
2. Then run the program

## Controls

**Left Player**

* W → Move Up
* S → Move Down

**Right Player**

* Up Arrow → Move Up
* Down Arrow → Move Down

## Technologies

* Python
* Turtle Graphics
* Object-Oriented Programming

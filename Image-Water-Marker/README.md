# ImageWatermarker

A simple **Python desktop app** built with **Tkinter** and **Pillow** to add a watermark to your images.

---

## Features

- Upload an image from your computer.
- Add a simple text watermark (`anand`) automatically.
- Saves the watermarked image as `watermarked_image.png`.
